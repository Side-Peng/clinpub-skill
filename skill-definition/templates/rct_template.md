# RCT 论文模板

> 遵循 CONSORT 报告规范，目标 SCI Q1/Q2 期刊水平

---

## Title

**规则**：PICOS 结构 — 研究对象(Patient)、干预(Intervention)、对照(Comparison)、结局(Outcome)、设计(Study design)

- 长度：15-25 字中文
- 必含：随机、盲法（如适用）、主要结局指标
- 示例："生活方式干预对轻度认知障碍患者认知功能的影响：一项随机对照试验"

---

## Abstract（结构化）

| 章节 | 内容 | 字数 |
|------|------|------|
| Background | 研究背景 + 目的（1-2句） | 50-80字 |
| Methods | 设计、场所、受试者、干预、主要结局、分析方法 | 100-150字 |
| Results | 主要结局效应量 + 95%CI + p 值，关键次要结局 | 100-150字 |
| Conclusions | 主要结论 + 临床意义 | 40-60字 |

- **最后写**，全文完成后再提炼

---

## 1. Introduction

### 结构（漏斗形）
```
① 疾病负担与研究背景（2-3句）
② 现有干预/治疗现状（2-3句）
③ 已知证据与局限性（关键Gap点）
④ 本研究的必要性和目的（PICOS 陈述）
```

### 写作要点
- 引用 3-5 篇高影响力文献，突出当前干预的证据缺口
- 明确交代**为什么需要这项 RCT**
- 最后一段以 "Therefore, we conducted a randomized controlled trial to..." 结束
- 第④部分陈述预先注册的假设

### 文献需求
- 流行病学数据（发病率、患病率）
- 现有干预方案的 meta 分析 / 系统评价
- 关键先行研究

## 2. Methods

### 2.1 Study Design and Participants
- 研究设计类型（平行/交叉/析因设计）
- 伦理审批信息 + 临床试验注册号（如 ChiCTRxxxxx）
- 受试者来源（门诊/社区/住院）
- **纳入标准**：具体列条
- **排除标准**：具体列条
- 招募时间和地点

### 2.2 Sample Size Calculation
- 假设的效应量（来自前期研究或文献）
- α 和 β 值（通常 α=0.05, β=0.20, power=0.80）
- 计算方法和软件（如 `pwr` 包的 `pwr.t.test()`）
- 考虑脱落率后的调整样本量
- **必须报告**：统计效力分析的详细参数

### 2.3 Randomization and Blinding
- 随机序列生成方法（计算机生成/随机数字表）
- 分配隐藏机制（密封不透光信封/中央随机）
- 盲法设计（单盲/双盲/开放标签 + 盲法评估）
- 揭盲流程

### 2.4 Intervention
- 干预组：具体方案（剂量、频率、疗程）
- 对照组：安慰剂/常规治疗/阳性对照
- 合并治疗的管理
- 依从性监测方法

### 2.5 Outcomes
- **主要结局**（一个）：精确定义，测量方法，测量时间点
- **次要结局**：列全但不超 5 个
- 探索性结局：标注其为探索性
- 安全性结局（如适用）
- 所有结局需有信度和效度引用

### 2.6 Statistical Analysis
- 统计软件及版本（如 R 4.3.2）
- 基线比较方法（连续变量：t 检验/Mann-Whitney；分类变量：卡方/Fisher）
- 主要分析：意向性分析(ITT)/符合方案集(PP)/修正ITT
- 主要结局比较方法（如协方差分析 ANCOVA）
- **效应量 + 95%CI** 报告要求
- 多重比较校正（FDR/Bonferroni）
- 亚组分析（标注为探索性）
- 缺失数据处理（多重插补 LOCF 等）

> **统计报告必须**：效应量 + 95%CI + 精确 p 值，假设检验结果（正态性、方差齐性）

## 3. Results

### 3.1 Participant Flow（Figure 1: CONSORT 流程图）
- 筛选数量 → 随机分配 → 干预 → 随访 → 分析
- 各阶段脱落人数及原因

### 3.2 Baseline Characteristics（Table 1）
- 按分组列基线特征
- 连续变量：Mean ± SD / Median (IQR)
- 分类变量：n (%)
- **不可进行组间基线差异的假设检验**（基线比较是描述性的）
- 格式：第一行为干预组和对照组，第一列为变量名

### 3.3 Primary Outcome
- 按 ITT 人群报告主要结局分析结果
- 效应量 + 95%CI + 精确 p 值
- 建议出图：**Figure 2**（森林图/箱线图/折线图）
- 补充 PP 分析（如适用）

### 3.4 Secondary Outcomes（Table 2）
- 列次要结局的效应量、95%CI、p 值
- 标注多重比较校正后的显著性
- 建议出图：**Figure 3**（次要结局森林图）

### 3.5 Subgroup Analysis（Figure 4: 亚组森林图）
- 亚组变量：年龄、性别、疾病严重程度等
- 交互作用 p 值标注
- 明确标注为探索性分析

### 3.6 Adverse Events（Table 3）
- 总 AE 发生率、严重 AE、因 AE 退出
- 按系统/器官分类

### 3.7 Supplementary Results
- 敏感性分析
- 依从性分析
- 其他预设的辅助分析

## 4. Discussion

### 结构（沙漏形 — 先具体再扩展）

```
① 主要发现总结（1-2句，先报告主要结局结果）
② 与既往研究的比较（2-3段，逐项对比）
③ 机制/解释（1段，why does this happen）
④ 临床意义（1段，so what）
⑤ 优势（1段，如随机化、盲法、样本量等）
⑥ 局限性（1段，样本量限制、单中心、随访时间等）
⑦ 结论（升华——对未来研究和临床实践的启示）
```

### 写作要点
- 先回答本研究问题，再与既往研究对比
- 不要过度重复 Results 中的数值
- 局限性必须诚实——审稿人更接受坦诚的局限
- 临床意义要有具体建议（而非空话）

---

## 5. Conclusions

- 1-2 句总结核心发现
- 1 句临床/研究启示
- **避免过度推论**（不说超出数据范围的话）
